# SalesPerformance_Excel
An interactive Excel dashboard to track and analyze sales performance, visualize key metrics, and provide actionable business insights.
Global Sales Dashboard — Excel Project

Overview:
This project contains a Global Sales Dashboard built in Microsoft Excel. It visualizes worldwide sales performance across regions, products, and time periods. The dashboard helps analyze key metrics using interactive charts, pivot tables, and slicers for insights.

Project Files:
Global_Sales_Dashboard.xlsx — Main Excel workbook containing:
* Raw sales data
* Pivot tables
* Interactive dashboard
* Charts and filters

Features:
* Interactive data visualization using Excel charts
* Pivot tables summarizing sales and profit
* Slicers for filtering by date, region, and product
* Easy exploration of trends and patterns

How to Use:
1. Open the Excel file (*Global_Sales_Dashboard.xlsx*) in Microsoft Excel.
2. Enable content if prompted for slicer and pivot interactivity.
3. Navigate to the Dashboard sheet.
4. Use the slicers to filter and explore data by year, region, or category.
5. Analyze charts and summary tables for insights.

Example Insights:
* Compare total sales across different regions
* Identify best-selling products and categories
* Track quarterly and yearly sales growth
* Spot seasonal trends and patterns

Future Improvements:
* Add geographical map visuals for regional sales
* Incorporate forecasting models (trend projections)
* Add key performance indicators (KPIs) for targets vs actuals
* Convert to Power BI for advanced analytics

Tools Used:
* Microsoft Excel
  – Pivot Tables
  – Charts & Graphs
  – Slicers
  – Conditional Formatting

Learning Outcomes:
By creating and using this dashboard, you will learn:
